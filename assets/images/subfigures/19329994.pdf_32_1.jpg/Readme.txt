This folder contains the segmentation result obtained by using the FigSplit system.

“*.jpg” is the panels segmented from the original submitted image.

“.txt” contains an M-by-4 matrix. Each row of the matrix indicates a location of a panel. Each row contains a vector in the format [x, y, width, height], where x and y correspond to the upper left corner of the panel, width and height indicate the width and the height of the panel.
